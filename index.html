Installing SQLScout

To install SQLScout on Android Studio or IntelliJ IDEA, please follow these steps:

  1. Open the "Settings" (on Windows and Linux) or "Preferences" (on Mac) dialog
  2. Select "Plugins" from the list on the left side
  3. Click "Install plugin from disk..."
  4. Select SQLScout.zip
  5. Restart the IDE

Getting Started

Once you installed SQLScout, connect to SQLite databases as follows.

Android devices

To connect to a database in an Android device (including the Android emulator), your application must be an Android application and it must be running on the emulator or a real device.

  1. Go to the "SQLite Explorer" view and click the "+" button on the view's toolbar
  2. Select "Android Data Source" from the drop-down
  3. In the "New Android SQLite Data Source" dialog, enter a descriptive name for the new data source
  4. Review default values (e.g. package name) and click "OK"

File System

To connect to a SQLite database in the file system, the application does not have to be an Android application.

  1. Go to the "SQLite Explorer" view and click the "+" button on the view's toolbar
  2. Select "Local Data Source" from the drop-down
  3. In the "New Local SQLite Data Source" dialog, click the "..." button and select the path of the SQLite database to open
  4. Give the new data source a descriptive name
  5. Optionally, click the "Test Connection" button to verify the selected file is a SQLite database Click "OK"

The new data source will appear in the "SQLite Explorer" view. More details about SQLScout's features can be found here.
