Installing SQLScout.

To install SQLScout on Android Studio or IntelliJ IDEA, please follow these steps:

  1. Open the "Settings" (on Windows and Linux) or "Preferences" (on Mac) dialog
  2. Select "Plugins" from the list on the left side
  3. Click "Install plugin from disk..."
  4. Select SQLScout.zip
  5. Restart the IDE
